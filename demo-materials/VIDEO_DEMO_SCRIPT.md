## ğŸ¥ Production Notes

### Recording Setup:
- **Screen Resolution:** 1920x1080 (Full HD)
- **Recording Software:** OBS Studio, Camtasia, or ScreenFlow
- **Audio:** Clear voiceover with minimal background noise
- **Mouse Cursor:** Visible with click animations

### Visual Style:
- Highlight important clicks with circles/arrows
- Use zoom-ins for text that's hard to read
- Add subtle transitions between scenes
- Consider picture-in-picture for presenter in corner

### Sample Emails to Prepare:
1. Customer inquiry (for response demo)
2. Project update with tasks (for task extraction)
3. Long technical email (for summary)
4. Meeting request (for meeting detection)

### Pre-Recording Checklist:
- [ ] Outlook open and signed in
- [ ] Email Assistant installed and working
- [ ] Sample emails created in test account
- [ ] Screen clean (close unnecessary programs)
- [ ] Audio tested
- [ ] Script printed for reference
- [ ] All features tested beforehand

### Editing Tips:
- Speed up loading/processing times slightly (1.5x)
- Add captions for accessibility
- Include B-roll of documentation files
- Add background music (subtle, professional)
- Export in multiple formats (MP4, WebM)

### Accessibility:
- Include closed captions
- Provide transcript (this script)
- Ensure good color contrast
- Keep text on screen long enough to read
